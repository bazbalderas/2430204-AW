Funciones

"Write less, do more"

¿Cómo se declara una función y cómo se llama en JavaScript?

function sumar(a, b) { return a + b; }  // declaración (se eleva)
const multiplicar = function(a,b){ return a*b; }; // expresión
sumar(2,3); // 5

¿Qué significa que una función tenga parámetros y un valor de retorno?

Los parámetros son entradas que una función recibe; el valor de retorno es la salida que da con return. Si no hay return, la función devuelve undefined.

¿Qué son las funciones flecha (=>) y qué ventajas tienen?

const sumar = (a,b) => a + b;      // sintaxis concisa
const obj = { n: 1, inc: () => this.n }; // `this` léxico (cuidado con objetos)

Ventajas: sintaxis más corta, this léxico osea que no se vuelve a enlazar, ideal para callbacks.

¿Qué es el scope en JavaScript y cómo afecta a las variables?

Es el ámbito en el que un nombre es visible.

  • Alcance global, de función y de bloque ({}).
  • let/const son de alcance de bloque, var es de alcance de función.
  • Los closures: las funciones internas capturan las variables externas.
