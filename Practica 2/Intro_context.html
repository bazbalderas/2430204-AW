Introducción y Contexto

"Write less, do more"

¿Qué es JavaScript y cuáles son sus principales usos en la web?

JavaScript es un lenguaje de programación que permite implementar funciones a las páginas web que van de la mano con HTML Y CSS, sus principales usos son hacer más dinámica una página web ya que amplian el rango y alcance de la página, agregando nuevas posibilidades a lo que se esta haciendo, también el comportamiento y manejo de eventos en la página.


¿En que año y con que proposito se creo JavaScript?

Fue creado en 1995 por Brendan Eich, pensando en lo repetitivas y aburridas que eran las páginas web quería agregar algo que pudiera hacerlas más llamativas de lo que eran, porque de que servía que tuvieran un buen diseño sin poder llevar a la página web al siguiente nivel..


¿Por qué JavaScript no es lo mismo que Java?

La primera diferencia notable es que Java al igual que muchos lenguajes, es un lenguaje compilado, JavaScript es interpretado y dinámico, y bueno lo más notable es que JavaScript esta dirigido a la rama de las páginas web


¿Cómo se complementa JavaScript con HTML y CSS?

Cada una de ellas tiene una tarea diferente, pongamos de ejemplo el cuerpo humano, HTML es el esqueleto, CSS es la piel y facciones y JavaScript son los movimientos y acciones que tiene el cuerpo.
