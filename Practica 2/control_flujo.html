Control de Flujo

"Write less, do more"

¿Cómo funcionan las sentencias condicionales if, else y switch?

Estas estructuras nos ayudan a poder darle un flujo al programa y cambiar lo que esta pasando dentro del mismo, osea es como darle la oportunidad que haga cosas diferentes segun lo que el usuario haga.

IF: Si se cumple una condición, se ejecuta un bloque de código. Si no, se puede ejecutar otro bloque con ELSE. Ejemplo: 

if (edad >= 18)

ELSE: Si la condición del IF no se cumple, se ejecuta este bloque. Ejemplo: 

else { alert('Menor de edad'); }

SWITCH: Permite evaluar una expresión y ejecutar diferentes bloques de código según el valor. Ejemplo: 

switch (nombre) {
  case 'Baz':
    alert('Es Baz :D');
    break;
  case 'Falter':
    alert('Es Falter :(');
    break;
  default:
    alert('No se quien eres');
}

¿Cuáles son las diferencias entre los bucles for, while y do...while?

IF: Se evalúa una condición y, si es verdadera, se ejecuta un bloque de código. Si es falsa, no se ejecuta nada.

WHILE: Se evalúa una condición antes de cada iteración. Si es verdadera, se ejecuta el bloque de código. Si es falsa, se sale del bucle.

DO WHILE: Similar al WHILE, pero se garantiza que el bloque de código se ejecute al menos una vez, ya que la condición se evalúa después de la ejecución.

¿Para qué sirven las palabras clave break y continue?

break: Sale del bucle (o de un switch) inmediatamente.

continue: Salta a la próxima iteración.

for (let i=0;i<10;i++){
  if (i===5) continue; // saltar 5
  if (i===8) break;    // detenerse en 8
}
