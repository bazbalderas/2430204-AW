¿Qué diferencia hay entre var, let y const?

La principal diferencia radica en el alcance (scope) y la mutabilidad:

  • var: Tiene un alcance global o de función, y se puede redeclarar y reasignar.
  • let: Tiene un alcance de bloque, no se puede redeclarar en el mismo bloque, pero sí se puede reasignar.
  • const: También tiene un alcance de bloque, no se puede redeclarar ni reasignar. Se utiliza para valores constantes.

Tipos de datos primitivos + ejemplos

  1. Number (Númerica): Representa tanto números enteros como decimales. Ejemplos: 1, 2, 3, 42, 3.14, -5, 0.5.
  2. String (Cadena): Representa texto o secuencias de caracteres. Ejemplos: "Hola Mundo!", 'JavaScript', `Template literal`.
  3. Boolean (Booleano): Representa valores de verdad, solo puede ser true (verdadero) o false (falso). Se usa principalmente en condiciones y lógica. Ejemplos: true, false.
  4. Null: Representa la ausencia intencional de cualquier valor. Es un valor especial que indica "nada" o "vacío". Ejemplo: null.
  5. Undefined: Indica que una variable ha sido declarada pero no se le ha asignado ningún valor. Ejemplo: undefined.
  6. Symbol: Representa un identificador único e inmutable. Útil para crear propiedades de objetos que no colisionen. Ejemplo: Symbol('descripcion').
  7. BigInt: Permite representar números enteros de precisión arbitraria. Ejemplo: 123456789012345678901234567890n.

¿Qué diferencia hayu entre un array y un objeto en JavaScript?

Aunque a simple vista podríamos decir que son lo mismo, cambian en algunas cosas, por ejemplo el array guarda valores numéricos en una lista ordenada, en cambio el objeto guarda en lo que se le conoce como un mapa clave, claves tipo string.

¿Qué son y cómo se usan los operadores aritméticos, lógicos y relacionales?

Operadores Aritméticos:

  • + (Suma): Suma dos números o concatena strings. Ejemplo: 5 + 3 = 8, "Hola" + " Mundo" = "Hola Mundo"
  • - (Resta): Resta el segundo número del primero. Ejemplo: 10 - 4 = 6
  • * (Multiplicación): Multiplica dos números. Ejemplo: 6 * 7 = 42
  • / (División): Divide el primer número por el segundo. Ejemplo: 15 / 3 = 5
  • % (Módulo): Devuelve el resto de la división. Ejemplo: 10 % 3 = 1
  • ** (Exponenciación): Eleva un número a la potencia de otro. Ejemplo: 2 ** 3 = 8
  • ++ (Incremento): Aumenta el valor en 1. Ejemplo: x++ o ++x
  • -- (Decremento): Disminuye el valor en 1. Ejemplo: x-- o --x

Operadores Lógicos:

  • && (AND lógico): Devuelve true solo si ambas condiciones son verdaderas. Ejemplo: true && false = false
  • || (OR lógico): Devuelve true si al menos una condición es verdadera. Ejemplo: true || false = true
  • ! (NOT lógico): Invierte el valor booleano. Ejemplo: !true = false, !false = true

Operadores Relacionales (Comparación):

  • == (Igualdad): Compara valores convirtiendo tipos si es necesario. Ejemplo: 5 == "5" = true
  • === (Igualdad estricta): Compara valores y tipos sin conversión. Ejemplo: 5 === "5" = false
  • != (Desigualdad): Verifica si los valores son diferentes. Ejemplo: 5 != 3 = true
  • !== (Desigualdad estricta): Verifica si valores o tipos son diferentes. Ejemplo: 5 !== "5" = true
  • > (Mayor que): Verifica si el primer valor es mayor. Ejemplo: 8 > 5 = true
  • < (Menor que): Verifica si el primer valor es menor. Ejemplo: 3 < 7 = true
  • >= (Mayor o igual que): Verifica si es mayor o igual. Ejemplo: 5 >= 5 = true
  • <= (Menor o igual que): Verifica si es menor o igual. Ejemplo: 4 <= 6 = true
