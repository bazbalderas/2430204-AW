¿Qué es el DOM y por qué es importante en el uso de JavaScript en páginas web?

El Document Object Model (DOM) es el árbol en memoria de tu página HTML. JS lo usa para leer y cambiar elementos y estilos, habilitando la verdadera interactividad.

document.querySelector('#msg').textContent = '¡Hola!';

¿Qué es un evento en JavaScript y menciona al menos 3 ejemplos comunes?

Un evento es una señal de que algo ha ocurrido (usuario o navegador).

  • click: cuando se hace clic en un elemento 
    boton.addEventListener('click', () => alert('¡Hola!'));
  • keydown: cuando se presiona una tecla 
    document.addEventListener('keydown', (e) => console.log(e.key));
  • submit: cuando se envía un formulario 
    form.addEventListener('submit', (e) => e.preventDefault());

¿Cuál es la diferencia entre alert, prompt y console.log?

  • alert(msg): popup que bloquea la ejecución y muestra texto.
  • prompt(msg): popup que devuelve el texto que el usuario escribe (o null).
  • console.log(valor): imprime en la consola del desarrollador (no bloquea, útil para depuración).

¿Cómo se abre la consola del navegador y qué ventajas tiene usarla para depurar código?

  • Chrome/Edge: Ctrl+Shift+I (Linux/Win) o Cmd+Option+I (Mac), luego pestaña Consola.
  • Firefox: Ctrl+Shift+K o Cmd+Option+K.

Es útil para ver logs/errores, ejecutar JS de forma interactiva, inspeccionar variables, probar fragmentos de código y depurar más rápido.
