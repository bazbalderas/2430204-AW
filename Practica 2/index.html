JavaScript

"Write less, do more"

Introducción y contexto

¿Qué es JavaScript y cuáles son sus principales usos en la web?

JavaScript es un lenguaje de programación que permite implementar funciones a las páginas web que van de la mano con HTML Y CSS, sus principales usos son hacer más dinámica una página web ya que amplian el rango y alcance de la página, agregando nuevas posibilidades a lo que se esta haciendo, también el comportamiento y manejo de eventos en la página.


¿En que año y con que proposito se creo JavaScript?

Fue creado en 1995 por Brendan Eich, pensando en lo repetitivas y aburridas que eran las páginas web quería agregar algo que pudiera hacerlas más llamativas de lo que eran, porque de que servía que tuvieran un buen diseño sin poder llevar a la página web al siguiente nivel..


¿Por qué JavaScript no es lo mismo que Java?

La primera diferencia notable es que Java al igual que muchos lenguajes, es un lenguaje compilado, JavaScript es interpretado y dinámico, y bueno lo más notable es que JavaScript esta dirigido a la rama de las páginas web


¿Cómo se complementa JavaScript con HTML y CSS?

Cada una de ellas tiene una tarea diferente, pongamos de ejemplo el cuerpo humano, HTML es el esqueleto, CSS es la piel y facciones y JavaScript son los movimientos y acciones que tiene el cuerpo.



Fundamentos del lenguaje

Sintaxis básica y comentarios

Muy parecida a la de otros lenguajes ya que se utilizan funciones, métodos, variables, etc. Pero tiene su propia gramática al momento de escribir su código. como aclaraciones, JavaScript es muy sensible a mayúsculas y minúsculas, siempre se procura mantener una buena práctica en la escritura del código. y sobre todo jamas olvidar el ;


Tipos de datos primitivos + ejemplos

  1. Number (Númerica): Representa tanto números enteros como decimales. Ejemplos: 1, 2, 3, 42, 3.14, -5, 0.5.
  2. String (Cadena): Representa texto o secuencias de caracteres. Ejemplos: "Hola Mundo!", 'JavaScript', `Template literal`.
  3. Boolean (Booleano): Representa valores de verdad, solo puede ser true (verdadero) o false (falso). Se usa principalmente en condiciones y lógica. Ejemplos: true, false.
  4. Null: Representa la ausencia intencional de cualquier valor. Es un valor especial que indica "nada" o "vacío". Ejemplo: null.
  5. Undefined: Indica que una variable ha sido declarada pero no se le ha asignado ningún valor. Ejemplo: undefined.
  6. Symbol: Representa un identificador único e inmutable. Útil para crear propiedades de objetos que no colisionen. Ejemplo: Symbol('descripcion').
  7. BigInt: Permite representar números enteros de precisión arbitraria. Ejemplo: 123456789012345678901234567890n.

Array vs Objeto

Aunque a simple vista podriamos decir que son lo mismo, cambian en algunas cosas, por ejemplo el array guarda valores númericos en una lista ordenada, en cambio el objeto guarda en lo que se le conoce como un mapa clave, claves tipo string.

Operadores (aritméticos, lógicos, relacionales)

Operadores Aritméticos:

  • + (Suma): Suma dos números o concatena strings. Ejemplo: 5 + 3 = 8, "Hola" + " Mundo" = "Hola Mundo"
  • - (Resta): Resta el segundo número del primero. Ejemplo: 10 - 4 = 6
  • * (Multiplicación): Multiplica dos números. Ejemplo: 6 * 7 = 42
  • / (División): Divide el primer número por el segundo. Ejemplo: 15 / 3 = 5
  • % (Módulo): Devuelve el resto de la división. Ejemplo: 10 % 3 = 1
  • ** (Exponenciación): Eleva un número a la potencia de otro. Ejemplo: 2 ** 3 = 8
  • ++ (Incremento): Aumenta el valor en 1. Ejemplo: x++ o ++x
  • -- (Decremento): Disminuye el valor en 1. Ejemplo: x-- o --x

Operadores Lógicos:

  • && (AND lógico): Devuelve true solo si ambas condiciones son verdaderas. Ejemplo: true && false = false
  • || (OR lógico): Devuelve true si al menos una condición es verdadera. Ejemplo: true || false = true
  • ! (NOT lógico): Invierte el valor booleano. Ejemplo: !true = false, !false = true

Operadores Relacionales (Comparación):

  • == (Igualdad): Compara valores convirtiendo tipos si es necesario. Ejemplo: 5 == "5" = true
  • === (Igualdad estricta): Compara valores y tipos sin conversión. Ejemplo: 5 === "5" = false
  • != (Desigualdad): Verifica si los valores son diferentes. Ejemplo: 5 != 3 = true
  • !== (Desigualdad estricta): Verifica si valores o tipos son diferentes. Ejemplo: 5 !== "5" = true
  • > (Mayor que): Verifica si el primer valor es mayor. Ejemplo: 8 > 5 = true
  • < (Menor que): Verifica si el primer valor es menor. Ejemplo: 3 < 7 = true
  • >= (Mayor o igual que): Verifica si es mayor o igual. Ejemplo: 5 >= 5 = true
  • <= (Menor o igual que): Verifica si es menor o igual. Ejemplo: 4 <= 6 = true

Control de Flujo

If / Else / Switch

Estas estructuras nos ayudan a poder darle un flujo al programa y cambiar lo que esta pasando dentro del mismo, osea es como darle la oportunidad que haga cosas diferentes segun lo que el usuario haga.

IF: Si se cumple una condición, se ejecuta un bloque de código. Si no, se puede ejecutar otro bloque con ELSE. Ejemplo: 

if (edad >= 18)

ELSE: Si la condición del IF no se cumple, se ejecuta este bloque. Ejemplo: 

else { alert('Menor de edad'); }

SWITCH: Permite evaluar una expresión y ejecutar diferentes bloques de código según el valor. Ejemplo: 

switch (nombre) {
  case 'Baz':
    alert('Es Baz :D');
    break;
  case 'Falter':
    alert('Es Falter :(');
    break;
  default:
    alert('No se quien eres');
}

if vs while vs do while

IF: Se evalúa una condición y, si es verdadera, se ejecuta un bloque de código. Si es falsa, no se ejecuta nada.

WHILE: Se evalúa una condición antes de cada iteración. Si es verdadera, se ejecuta el bloque de código. Si es falsa, se sale del bucle.

DO WHILE: Similar al WHILE, pero se garantiza que el bloque de código se ejecute al menos una vez, ya que la condición se evalúa después de la ejecución.

break & continue

break: Sale del bucle (o de un switch) inmediatamente.

continue: Salta a la próxima iteración.

for (let i=0;i<10;i++){
  if (i===5) continue; // saltar 5
  if (i===8) break;    // detenerse en 8
}

Funciones

Declarar & llamar

function sumar(a, b) { return a + b; }  // declaración (se eleva)
const multiplicar = function(a,b){ return a*b; }; // expresión
sumar(2,3); // 5

Parámetros & valor de retorno

Los parámetros son entradas que una función recibe; el valor de retorno es la salida que da con return. Si no hay return, la función devuelve undefined.

Funciones flecha (=>) & ventajas

const sumar = (a,b) => a + b;      // sintaxis concisa
const obj = { n: 1, inc: () => this.n }; // `this` léxico (cuidado con objetos)

Pros: sintaxis más corta, this léxico (no se vuelve a enlazar), ideal para callbacks.

Limitaciones: no son constructores, no tienen arguments/prototype.

Scope (alcance)

Es el ámbito en el que un nombre es visible.

  • Alcance global, de función y de bloque ({}).
  • let/const son de alcance de bloque, var es de alcance de función.
  • Los closures: las funciones internas capturan las variables externas.

Objetos y colecciones

Declarar objeto & acceder a propiedades

const usuario = { nombre: 'Ana', edad: 20, 'color-favorito': 'azul' };
usuario.nombre         // con punto
usuario['color-favorito'] // con corchetes (útil para claves dinámicas o no válidas)

Propiedad vs Método

Una propiedad es un dato (usuario.edad). Un método es una función almacenada en un objeto:

const temporizador = {
  inicio: 0,
  resetear(){ this.inicio = Date.now(); } // método
};

Métodos básicos de los arrays (al menos 3)

  • push(item): añade al final; devuelve el nuevo tamaño.
  • pop(): elimina el último; devuelve el elemento eliminado.
  • shift() / unshift(item): elimina/agrega al principio.
  • map(fn): devuelve un nuevo array transformando cada elemento.
  • filter(fn): devuelve un nuevo array con elementos que cumplen una condición.
  • reduce(fn, init): reduce el array a un solo valor.

Ejemplo:

[1,2,3].map(x=>x*2)      // [2,4,6]
[1,2,3,4].filter(x=>x%2) // [1,3]
[1,2,3].reduce((s,x)=>s+x,0) // 6

Otros temas básicos

¿Qué es el DOM y por qué es importante?

El Document Object Model (DOM) es el árbol en memoria de tu página HTML. JS lo usa para leer y cambiar elementos y estilos, habilitando la verdadera interactividad.

document.querySelector('#msg').textContent = '¡Hola!';

¿Qué es un evento? (3 ejemplos)

Un evento es una señal de que algo ha ocurrido (usuario o navegador). Ejemplos: click, input, submit, keydown, change, load.

boton.addEventListener('click', () => alert('¡Hola!'));

¿Cuál es la diferencia entre alert, prompt y console.log?

  • alert(msg): popup que bloquea la ejecución y muestra texto.
  • prompt(msg): popup que devuelve el texto que el usuario escribe (o null).
  • console.log(valor): imprime en la consola del desarrollador (no bloquea, útil para depuración).

¿Cómo se abre la consola del navegador y qué ventajas tiene?

  • Chrome/Edge: Ctrl+Shift+I (Linux/Win) o Cmd+Option+I (Mac), luego pestaña Consola.
  • Firefox: Ctrl+Shift+K o Cmd+Option+K.

Es útil para ver logs/errores, ejecutar JS de forma interactiva, inspeccionar variables, probar fragmentos de código y depurar más rápido.
